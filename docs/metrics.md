# Métricas

## Fluxo

| Métrica | Meta |
|---------|------|
| Lead Time (criação da issue → merge) | ≤ 5 dias úteis |
| Cycle Time (início do trabalho → merge) | ≤ 3 dias úteis |
| Throughput | ≥ 3 issues/sprint |
| WIP por pessoa | ≤ 2 |

## Qualidade

| Métrica | Meta |
|---------|------|
| Bugs críticos por sprint | ≤ 1 |
| PRs rejeitados em review | ≤ 30% |
| Requisitos com critério de aceite testável | 100% |

## Coleta

Métricas coletadas a cada review de sprint (sexta-feira) e revisadas a cada marco do PI.
